# AI 摄影评价 — 使用指南

## 一、简介

AI 摄影评价是一款在浏览器中直接运行的摄影作品分析工具。你只需上传照片、填入自己的 API Key，即可调用 Moonshot AI 或 OpenAI 的视觉模型，从构图、光影、色彩、叙事、后期等维度获得结构化评价。

**核心理念**：AI 能识别图像中的模式，却无法拥有人类的凝视；真正的审美，永远属于按下快门的那个人。

---

## 二、快速开始

### 1. 打开页面
直接用浏览器打开 `index.html`（无需安装，无需后端）。

### 2. 上传照片
- 点击上传区域，或把图片拖拽进虚线框
- 支持 JPG、PNG、WebP，建议小于 20MB
- 大于 25MB 的图片会自动压缩后上传

### 3. 选择模型并填入 API Key
| 提供商 | 可选模型 |
|--------|----------|
| Moonshot AI | kimi-k2.6、kimi-k2.5、moonshot-v1-8k/32k/128k-vision |
| OpenAI | gpt-4o、gpt-4o-mini |

在「API Key」栏填入你自己的 Key（`sk-...`）。**分析将直连 AI 服务商，Key 仅在本地使用，不会上传给任何第三方服务器。**

### 4. 选择分析模式
- **🌱 入门模式**：温暖易懂，像朋友一样发现照片的美好，给出 1-2 条简单改进建议
- **📷 专业模式**：深度技术解析，含评分环、分项得分、详细优缺点与可执行修改方案

### 5. 点击「开始分析」
等待约 10-60 秒（取决于模型和图片大小），结果将出现在右侧。

## 三、功能详解

### 3.1 EXIF 信息自动读取
上传 JPEG 后，工具会在本地解析拍摄参数并展示为标签：
- 相机型号、镜头、光圈、快门、ISO、焦距
- 拍摄时间、测光模式、曝光程序、GPS 等

&gt; 这些信息会一并送入 AI 提示词，帮助模型结合拍摄策略做更精准的技术点评。

### 3.2 入门模式结果
- **总体寄语**：一句温暖、自然的整体评价
- **值得表扬**：最多 3 条优点，用通俗语言说明好在哪里
- **小建议**：最多 3 条新手友好的改进方向

### 3.3 专业模式结果
- **综合评分环**：0-10 分，带动画进度
- **照片类型与风格标签**：如「风光」「街头」「极简」等
- **分项得分卡**（6 项）：
  - 构图、光影、色彩、叙事、后期、综合
  - 每项带独立进度条
- **五大维度分析卡片**：
  - 每张卡片含：深度分析、优点、可优化点、修改建议
  - 卡片左侧带颜色标识（构图-蓝、光影-橙、色彩-红、叙事-紫、后期-绿）

### 3.4 保存长图
分析完成后，点击「保存长图」按钮，即可将：
- 页面标题与链接
- 原图预览
- 全部分析结果

生成一张 800px 宽、2 倍高清的 PNG 长图，方便分享到朋友圈、小红书、摄影社群。

&gt; 保存时会强制使用浅色主题，确保在任何设备上阅读效果一致。

## 四、评分标准说明

为了让评分真正具有参考价值，AI 被明确要求**拉开区分度**，避免所有照片都挤在 6-8 分。

| 分数段 | 含义 |
|--------|------|
| **1-3 分** | 基础缺失或严重失误（主体模糊、严重过曝/欠曝、构图混乱、无主题） |
| **4-5 分** | 有明显不足，爱好者普通水平（光线平淡、构图随意、色彩失调） |
| **6 分** | 及格线，基本完整但缺乏亮点，看完不留印象 |
| **7 分** | 良好，有可取之处，仍有明显优化空间 |
| **8 分** | 优秀，完成度高，至少一个维度表现突出 |
| **9 分** | 出色，接近专业水准，多维度协调统一，具视觉冲击力 |
| **10 分** | 卓越，具有艺术感染力或技术完美，可获奖/展览水准 |

**评分原则**：
- 普通随手拍给 4-6 分是合理的
- 有明显硬伤的作品不应高于 5 分
- 真正优秀的作品不要吝啬 9-10 分
- 各维度允许拉开差距（如构图 9 分、后期 5 分）
- 综合评分不是平均分，可偏向最强或最弱维度

## 五、深色模式

工具自动跟随系统主题：
- **浅色**：Apple 风格明亮界面，蓝白主调
- **深色**：深蓝灰背景，降低亮度，适合夜间使用

&gt; 保存长图时始终使用浅色主题，保证分享后的可读性。

## 六、常见问题

**Q：API Key 安全吗？**  
A：Key 仅在浏览器本地使用，直接发往你选择的 AI 服务商（Moonshot/OpenAI），不会经过任何中间服务器。

**Q：为什么分析失败？**  
常见原因：
- **401**：API Key 无效或过期
- **429**：请求太频繁或额度用尽
- **超时（5 分钟）**：网络不稳定或图片过大，建议换小图重试
- **JSON 格式异常**：当前模型不支持强制 JSON 输出，建议换 kimi-k2.5 / gpt-4o

**Q：为什么评分比预期低/高？**  
A：这是好事——说明评分真正拉开了区分度。AI 已被告知要客观诚实，不再做「老好人」。如果认为某次评分不公，可以换一张图或换一个模型交叉验证。

**Q：可以离线使用吗？**  
A：不可以。分析需要调用云端大模型 API，必须联网。
