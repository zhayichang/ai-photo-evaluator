# AI 摄影作品评价 — 使用说明

## 简介

这是一款基于 AI 视觉大模型的摄影作品评价工具。上传照片后，AI 会从构图、光影、色彩、叙事等多个维度进行专业或友好的分析，并生成可保存分享的长图。

浏览器不会保存或接触 Moonshot API Key。图片先在本地压缩，再通过本站安全代理转发至 Moonshot，本站不长期保存图片。

---

## 快速开始（三步搞定）

### 步骤 1：上传照片
- **点击上传区域**或**拖拽图片**到「上传照片」卡片内。
- 支持格式：JPG、PNG、WebP。
- 所有图片发送前都会自动优化：最长边限制为 1920 像素，并动态调整 JPEG 质量与尺寸，尽量控制在 5 MB 以内。
- 上传后，若照片包含 EXIF 信息，会自动显示拍摄参数（相机、镜头、光圈、快门、ISO 等）。


### 步骤 2：选择分析模式
| 模式 | 适合人群 | 分析特点 |
|---|---|---|
| **🌱 入门模式** | 摄影新手 | 语言温暖易懂，以鼓励和发现优点为主，给出 1-2 个简单可执行的建议 |
| **📷 专业模式** | 进阶/专业用户 | 深度技术解析，含 0-10 分维度评分（构图、光影、色彩、叙事、综合），并附详细改进建议 |

点击模式卡片即可切换，按钮文字会随模式自动变化。

### 步骤 3：开始分析
点击「开始友好点评」或「开始专业分析」按钮，等待 AI 返回结果。
- 分析过程通常需要 10-60 秒，取决于网络与模型响应速度。
- 服务端默认使用 `kimi-k2.6`，服务异常时最多尝试一个后备模型。
- 浏览器只上传一次图片；模型选择、提示词和 API Key 均保留在服务端。
- 上传阶段超时为 45 秒，AI 分析总超时为 180 秒。

## 结果解读

### 入门模式结果
- **总体寄语**：AI 对这张照片的整体感受（温暖风格）。
- **值得表扬**：列出 3 个最突出的优点。
- **小建议**：给出 2-3 条简单、好上手的改进方向。

### 专业模式结果
- **综合评分卡片**：左上角显示照片类型与一句话总结，右侧为 0-10 分综合评分圆环。
- **风格标签**：自动识别摄影风格（如人文纪实、街头、风光等）。
- **维度评分网格**：构图、光影、色彩、叙事、综合，每项均配进度条。
- **分析卡片**（按维度展开）：
  - **分析**：该维度的整体解读
  - **优点**：作品值得肯定的地方
  - **可优化点**：当前存在的不足
  - **修改建议**：具体、可操作的提升方案

## 保存与分享

分析完成后，点击「保存长图」按钮即可将评价结果导出为 PNG 高清长图（2 倍分辨率），方便发朋友圈、小红书或摄影社群。

&gt; **提示**：导出前请确保当前显示的是你需要的模式（专业/入门），截图会按当前可见内容生成。

## 常见问题

**Q：分析失败了怎么办？**
A：常见原因及对策：
- **413 错误**：图片超过服务端限制，请换一张尺寸更小的图片。
- **429 错误**：当前网络地址或全站当天请求较多，请稍后再试。
- **安全验证失败**：重新完成人机验证后提交。
- **上传超时**：切换到更稳定的网络后重试。
- **AI 超时**：服务繁忙，请稍后重试；重复请求会短时间去重。

**Q：EXIF 信息没有显示？**
A：仅 JPG 格式支持 EXIF 提取；若照片经过微信、微博等社交平台压缩，EXIF 可能已被抹除。

**Q：支持手机使用吗？**
A：支持。页面采用响应式设计，在手机上会自动调整为单栏布局，操作流畅。

## 隐私说明

- 照片在浏览器本地压缩后，以二进制形式发送至本站安全代理，再由代理转发至 Moonshot。
- 本站不长期保存图片；短时间请求去重只缓存 AI 文本结果。
- GPS、拍摄者姓名和设备序列号不会发送给代理。
- API Key 只配置在服务端环境变量中，不会进入浏览器或前端构建产物。
- 为执行限流和排查故障，服务端会短期处理请求 ID、网络地址、耗时、状态码、模型和图片大小，不记录图片内容。

## 技术栈

- 前端：原生 HTML / CSS / JavaScript
- 安全代理：Node.js / 阿里云函数计算
- 限流与去重：Redis
- 截图导出：html2canvas
- AI 接口：Moonshot AI Chat Completions API
- EXIF 解析：纯前端轻量 JPEG 解析器（无第三方库）
