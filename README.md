## 🚀 快速开始

1. 克隆或下载本项目三个文件到同一目录
2. 直接用浏览器打开 `index.html`（无需构建工具）
3. 在页面中填入你自己的 API Key
4. 上传照片，选择模式，点击开始分析

> **注意**：由于浏览器安全策略（CORS），直接通过 `file://` 协议打开时 API 请求可能会被拦截。建议通过本地服务器访问，例如：
> ```bash
> npx serve .
> # 或
> python -m http.server 8080
> ```

## 🔧 配置说明

### API Key

在页面第二步中填入你的 API Key：

| 提供商 | API Key 格式 | 获取地址 |
|--------|-------------|---------|
| Moonshot AI | `sk-...` | [platform.moonshot.cn](https://platform.moonshot.cn) |
| OpenAI | `sk-...` | [platform.openai.com](https://platform.openai.com) |

**安全提示**：API Key 仅存储在浏览器本地（页面刷新后需重新输入），所有请求均从前端直接发往对应官方接口，不会经过任何中间服务器。

### 支持模型

- **Moonshot AI**：`kimi-k2.6`、`kimi-k2.5`、`moonshot-v1-8k-vision-preview`、`moonshot-v1-32k-vision-preview`、`moonshot-v1-128k-vision-preview`
- **OpenAI**：`gpt-4o`、`gpt-4o-mini`

## 🎨 两种分析模式

### 入门模式
- 语气像朋友一样温暖、易懂
- 自动提炼 3 个最值得表扬的优点
- 给出 1~2 个对新手非常友好的、可立即执行的小建议
- 避免过多专业术语，以鼓励和发现美好为主

### 专业模式
- 结构化输出：构图、光影、色彩、叙事、后期五大维度
- 每项维度包含：深度分析、优点、可优化点、具体修改建议
- 综合评分（0–10 分）与分项评分，附带可视化评分圆环与进度条
- 额外提供视觉焦点、景深层次、视觉动线、情绪基调、风格参考等进阶分析
- 评价客观克制，优先肯定作品已成立的部分，再给出建设性优化意见

## 📷 EXIF 信息支持

上传 JPEG 照片后，工具会在本地解析以下 EXIF 数据并展示：

| 参数 | 说明 |
|------|------|
| 相机 / 型号 | 机身与型号信息 |
| 镜头 | 镜头厂商与型号 |
| 光圈 | 如 `f/2.8` |
| 快门速度 | 如 `1/125s` |
| ISO | 感光度 |
| 焦距 | 实际焦距与 35mm 等效焦距 |
| 分辨率 | 图像宽高 |
| 拍摄时间 | 原始拍摄日期 |
| 拍摄模式 | 手动、光圈优先、快门优先等 |
| 测光模式 | 平均、点测、中央重点等 |
| 闪光灯 | 是否触发 |
| GPS 坐标 | 经纬度（如有） |
| 白平衡 / 方向 / 软件 | 其他技术参数 |

## ⚠️ 已知限制

- **CORS 限制**：直接打开本地 HTML 文件时，跨域请求可能被浏览器阻止，建议使用本地服务器运行
- **JSON 模式**：OpenAI 端使用 `response_format: { type: "json_object" }` 强制 JSON 输出；Moonshot 端依赖模型遵循 Prompt 中的 JSON 格式要求
- **超时时间**：单次分析请求最长等待 5 分钟（300 秒），超大图片或复杂场景可能需要更久
- **图片格式**：主要支持 JPG、PNG、WebP；EXIF 解析仅对 JPEG 有效
