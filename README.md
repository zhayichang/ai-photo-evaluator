# AI 摄影作品评价工具

## 简介

**AI 摄影作品评价** 是一个基于浏览器的智能摄影分析工具，通过调用 Moonshot AI（Kimi）或 OpenAI 的视觉大模型，为你的摄影作品提供专业或友好的深度点评。

无论你是想获得专业级的技术解析，还是希望得到温暖易懂的入门建议，这款工具都能满足你的需求。所有分析直接通过你自己的 API Key 直连 AI 服务商，**数据安全，隐私可控**。

---

## 功能亮点

- 🖼️ **智能图像分析**：支持 JPG、PNG、WebP 格式，自动压缩大图，快速上传
- 📊 **双模式评价**：
  - **入门模式**：温暖鼓励，通俗易懂，适合摄影新手
  - **专业模式**：深度技术解析，多维度评分，适合进阶爱好者和摄影师
- 📷 **EXIF 信息读取**：自动提取相机型号、光圈、快门、ISO、焦距等拍摄参数，辅助 AI 更精准分析
- 📈 **可视化评分**：专业模式提供 0-10 分制的多维度雷达评分（构图、光影、色彩、叙事、后期）
- 💾 **一键保存长图**：将评价结果生成精美长图，方便分享到朋友圈、小红书等平台
- 🌙 **深色模式适配**：自动跟随系统深色模式，保护眼睛
- 🔒 **隐私安全**：API Key 仅存储在本地浏览器，所有请求直连服务商，不经过第三方服务器

## 快速开始

### 1. 准备工作

打开工具页面，你会看到左右分栏的布局：**左侧为操作区，右侧为结果展示区**。

### 2. 上传照片

在左侧第一个卡片中：
- **点击上传**：点击虚线框区域，从电脑中选择照片
- **拖拽上传**：直接将图片拖拽到上传区域

支持格式：JPG、PNG、WebP（建议小于 20MB，超大图片会自动压缩）

上传成功后，会显示图片预览，并自动读取 EXIF 拍摄参数（如相机型号、镜头、光圈、快门等信息）。

### 3. 配置 AI 模型

在第二个卡片中：
- **选择 API 提供商**：Moonshot AI（推荐）或 OpenAI
- **选择模型**：
  - Moonshot：kimi-k2.6、kimi-k2.5、moonshot-v1-8k/32k/128k-vision
  - OpenAI：gpt-4o、gpt-4o-mini
- **输入 API Key**：在输入框中填入你的 API Key（格式如 `sk-...`）

&gt; 💡 **如何获取 API Key？**
&gt; - **Moonshot AI**：访问 [platform.moonshot.cn](https://platform.moonshot.cn) 注册并创建 API Key
&gt; - **OpenAI**：访问 [platform.openai.com](https://platform.openai.com) 获取 API Key

### 4. 选择分析模式

在第三个卡片中选择评价风格：

| 模式 | 适合人群 | 特点 |
|------|----------|------|
| 🌱 **入门模式** | 摄影新手 | 语言温暖易懂，以鼓励和发现优点为主，给出简单好上手的改进建议 |
| 📷 **专业模式** | 进阶爱好者/摄影师 | 深度技术解析，多维度评分（构图、光影、色彩、叙事、后期），建设性批评 |

### 5. 开始分析

点击底部蓝色按钮 **"开始友好点评"** 或 **"开始专业分析"**。

等待 10-60 秒（取决于网络和图片复杂度），AI 正在分析你的作品...

### 6. 查看结果

分析完成后，右侧会展示完整的评价报告：

**入门模式结果**：
- ✨ 总体鼓励语
- 👍 值得表扬的优点（最多 3 条）
- 💡 简单实用的小建议（最多 3 条）

**专业模式结果**：
- 📋 照片类型识别与风格标签
- 🎯 综合评分圆环（0-10 分）
- 📊 五维评分卡片（构图、光影、色彩、叙事、后期）
- 📝 详细分析卡片（每个维度包含分析、优点、可优化点、修改建议）
- 🔍 进阶分析（视觉焦点、景深层次、视觉流动、情绪基调、风格参考）

### 7. 保存分享

点击结果顶部的 **"保存长图"** 按钮，即可将评价结果生成一张精美的长图（包含原图和评价内容），方便保存到本地或分享到社交媒体。
