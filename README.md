# AI 摄影评价

一个轻量级的浏览器端摄影作品智能评价工具，采用纯 `HTML/CSS/JavaScript` 实现。用户可以上传照片，读取本地 EXIF 数据，并通过 AI 模型对作品进行入门友好点评或专业分析。

## 主要功能

- 图片上传与本地预览
- JPEG EXIF 元数据提取（支持拍摄时间、光圈、快门、ISO、焦距、机型与 GPS 等）
- 导入 AI API Key 并选择 AI 服务提供商与模型
- 入门模式与专业模式两种评价风格
- 右侧结果面板支持默认提示、加载状态和分析结果展示
- 专业模式输出评分、优点、可优化点与结构化分析
- 入门模式输出鼓励寄语、值得表扬点和小建议

## 文件结构

- `index.html` - 页面结构与内容
- `style.css` - 界面样式
- `script.js` - 图片处理、EXIF 解析和 AI 交互逻辑
- `README.md` - 项目说明文档

## 使用方式

1. 打开 `index.html`。
2. 选择或拖拽上传照片。
3. 在左侧设置区：
   - 选择 API 提供商
   - 选择模型
   - 输入 API Key
4. 选择分析模式：`入门模式` 或 `专业模式`。
5. 点击“开始分析”。
6. 右侧卡片会显示等待提示、加载状态，并最终呈现 AI 分析结果。


## 说明

- 该项目依赖用户自己的 AI API Key 进行在线分析。
- 默认支持 Moonshot AI 与 OpenAI 两类服务提供商，且可以选择不同模型。
- EXIF 信息仅在 JPEG 图片中进行本地解析。
